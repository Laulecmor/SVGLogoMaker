## Installation Instructions
- Clone the repository 
- Install Node.js
- Install Jest
- Install inquiere.js 

## Usage
- In VS Code, navigate to the index.js file and open an integrated terminal
- Type "node index.js"
- Follow the prompts to design your Logo
